# Arthur Tasks

Issue-driven Agent Orchestration for OpenClaw

## Labels

- `agent:*` - Route to specific agent
- `status:*` - Task status
- `priority:*` - Priority level

## Workflow

1. Create Issue with task description
2. Add agent label (agent:coder, agent:researcher, etc.)
3. Agent picks up and executes
4. Results posted as comment
5. Issue closed
